# CRM Dashboard

A simple CRM / Client Management Dashboard UI built with **HTML, CSS, and JavaScript**.  
This project demonstrates a basic dashboard layout for small agencies or B2B service businesses.

## 🚀 Features
- ✅ Client list management  
- ✅ Lead pipeline stages (New → Contacted → Proposal → Closed)  
- ✅ Task tracker & follow‑up reminders  
- ✅ Lightweight, responsive design  

## 📂 Tech Stack
- HTML for structure  
- CSS for styling  
- JavaScript for simple interactivity
